# Requisitos

Antes de instalar o sistema, certifique-se de que os seguintes softwares estão instalados em seu computador.

## Node.js

O Node.js é necessário para executar o sistema de visualização.

**Versão testada:** 24.13.0

Download oficial:
https://nodejs.org/

---

## Python

O Python é necessário para executar o aplicativo de terminal.

**Versão testada:** 3.12.9

Download oficial:
https://www.python.org/downloads/

---

## MySQL Community Server

O MySQL é utilizado para armazenar os dados importados do altímetro.

**Versão testada:** 9.7

Download oficial:
https://dev.mysql.com/downloads/mysql/
